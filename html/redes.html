MODELO OSI




El Modelo OSI es una referencia conceptual creada por la Organización Internacional de Normalización (ISO) en 1984. Este modelo divide las funciones de la comunicación en redes en siete capas jerárquicas, cada una con responsabilidades específicas. Es principalmente un modelo teórico utilizado para entender y diseñar sistemas de redes.



Las siete capas del Modelo OSI:

Capa Física
Define las características físicas del hardware de red, como cables, conectores y señales eléctricas. Ejemplo: Ethernet, USB.

Capa de Enlace de Datos
Garantiza una comunicación libre de errores entre nodos directamente conectados. Aquí se encuentran protocolos como Ethernet y Wi-Fi.

Capa de Red
Se encarga del enrutamiento y la asignación de direcciones lógicas (IP). Protocolos: IP (IPv4, IPv6).

Capa de Transporte
Asegura la transmisión fiable de datos entre sistemas finales. Protocolos: TCP, UDP.

Capa de Sesión
Establece, administra y termina sesiones entre aplicaciones. Es menos utilizada directamente en redes modernas.

Capa de Presentación
Traduce los datos al formato que entienden las aplicaciones (por ejemplo, cifrado, compresión). Ejemplo: SSL/TLS.

Capa de Aplicación
Permite la interacción directa con los usuarios finales a través de software. Protocolos: HTTP, FTP, SMTP.

Ventajas del Modelo OSI:

Estandarización de procesos de red.
Separación clara de funciones, facilitando la comprensión y solución de problemas.
Flexibilidad para el desarrollo de nuevas tecnologías.



MODELO TCP/IP




El Modelo TCP/IP, desarrollado en los años 70 por el Departamento de Defensa de los Estados Unidos, es el conjunto de protocolos utilizado en la práctica para la mayoría de las comunicaciones en redes, especialmente en Internet. Es un modelo más simple que el OSI, compuesto por cuatro capas principales, que en esencia agrupan las funciones del Modelo OSI.


Las cuatro capas del Modelo TCP/IP:

Capa de Acceso a la Red
Combina las funciones de las capas física y de enlace del Modelo OSI. Aquí se definen las tecnologías de acceso, como Ethernet y Wi-Fi.

Capa de Internet
Se ocupa del direccionamiento y enrutamiento, utilizando el protocolo IP como núcleo.

Capa de Transporte
Proporciona un intercambio fiable o rápido de datos, según el protocolo utilizado (TCP o UDP).

Capa de Aplicación
Incluye todos los protocolos de alto nivel como HTTP, SMTP y DNS, permitiendo a las aplicaciones interactuar directamente con los usuarios.

Ventajas del Modelo TCP/IP:

Está diseñado para ser práctico y eficiente.
Es la base de Internet, garantizando su interoperabilidad global.
Soporta una amplia variedad de tecnologías y dispositivos.
